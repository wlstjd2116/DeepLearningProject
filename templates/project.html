LSTM 알고리즘을 활용한 주가 예측 프로그램이란?

본 프로젝트는 주가 예측을 위해 딥러닝으로 구현됐습니다.



딥러닝에 사용된 데이터 종류

  • 여러 변수에 의한 주가 데이터 예측
  • 'Petronet'에서 제공하는 국제유가 데이터 [Brent, WTI, Dubai, Oman]
  • 'Investing.com'의 미국 달러/원 환율 데이터 [시가, 종가, 고가, 저가]
  • 질병관리청에서 제공하는 코로나 바이러스 확진자 데이터

데이터 수집 기간

2021년 6월 25일 ~ 2022년 7월 15일
(일 단위로 수집, 각 특성마다 261개의 데이터를 딥러닝에 활용)
